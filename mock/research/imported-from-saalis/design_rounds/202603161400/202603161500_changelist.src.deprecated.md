## `saalis-sdk/src/lib.rs`

Core trait definitions. This is the largest file — all SDK contracts.
Organised by module but initially in one file for simplicity:

- `Named`, `Registrable`, `HasDescriptor<D>`
- `Descriptor`, `Registry`
- `SdkVersion`
- `MetadataTable`, `Column<I, A>`, `In<M>`, `As<V>`, `References<T>`
- `ColumnValue`, `SqlAffinity` (Integer, Real, Text, Blob)
- `Migration<V>`, `MigrationOpSet`
- `EntityType`, `SupportsEntity<E>`, `HasMetadata<M>`
- `Connector`, `Cataloguer<E>`, `Enricher<E>`, `Sourcer<E, S>`
- `SourceType`, `SourceUrl`
- `WorkUnit<F, W, R>`, `For<T>`, `With<I>`, `Returns<O>`
- `ForTarget`, `WithInput`, `ReturnsOutput`
- `EventKind`, `EventKindSet`
- `Subsystem`
- `HttpMethod`, `HttpClient`, `Get`, `Post`, `Put`, `Delete`, `Patch`, `Head`
- `Storage`, `FileStorage`, `ManagedStorage`
- `StorageLocation`, `Managed`, `Redirectable`
- `LocationHandle<L, S>`
- `ConnectorContext<C>`, `SchedulerContext`
- `QueryBuilder<M>`, `WriteBuilder<M>`
- `QueryFilter`, `CompositeField`
- `NotificationPayload`
- `SettingsValue`, `SettingsDomain`, `SetAccess<R, A>`, `ResolveAccess<R>`
- `AccessLevel`, `Hidden`, `ReadOnly`, `ReadWrite`
- `Credential`, `Secret`, `Locale`
- `Role`, `Admin`, `User`, `Child`
- `ErrorKind`, `Transient`, `Fixable`, `Permanent`
- `AppError<K>`
- `Theme`, `ThemeProperty`, `Provides<P, G>`, `AsVariantOf<T>`
- `ThemeTranslator<F>`, `ThemeBuilder<F>`, `Fragment<F>`
- `Str`, `EntityId`, `ExternalId<C>`, `Port`, `RateLimit`

All trait methods use `todo!()` or are marker traits with no methods.
Struct fields use placeholder types. The goal is COMPILABLE TYPE
SIGNATURES, not implementations.

1. Infrastructure: delete core/build.rs, create primitives/build.rs
2. `saalis-primitives/src/lib.rs` (leaf, no deps)
3. `saalis-derive/src/lib.rs` (depends on primitives)
4. `saalis-sdk/src/lib.rs` (depends on derive + primitives)
5. `saalis-scheduler/src/lib.rs` (depends on sdk)
6. `saalis-persistence/src/lib.rs` (depends on sdk + scheduler)
7. `saalis-core/src/lib.rs` (depends on sdk + scheduler + persistence)
8. `saalis-web/src/lib.rs` (depends on core)
9. Extension stubs (depend on sdk only)
10. `cargo check` — must compile
11. `cargo mock` — regenerate docs with populated STRUCTURE.md
