## `mock/crates/hilavitkutin-str/DESIGN.md.tmpl`

Update the `Str` description:

- `pub struct Str(Bits<32>)` (private inner field). Use `arvo_bits::Bits<32>` for the packed bit-layout.
- Layout unchanged: bit 31 origin flag, bits 30-28 reserved, bits 27-0 ID.
- Masks (`CONST_ORIGIN_FLAG`, `RUNTIME_MASK`, `ID_MASK`) promoted to `Bits<32>`.
- Accessor API unchanged: `is_const()`, `is_runtime()`, `id()`.
- New crate-level dependency on `arvo-bits`.
- Note that a dedicated `Bitfield` primitive is a better long-term fit (arvo task #127); today Str uses `Bits<32>`.
