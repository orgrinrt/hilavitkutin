### Fix 1 — `src/handle.rs` (hilavitkutin-str): rename CONST_MASK

Before:

```rust
/// Mask for the const-origin bit (bit 31 = 0).
pub const CONST_MASK: u32 = 0 << 31;
/// Mask for the runtime-origin bit (bit 31 = 1).
pub const RUNTIME_MASK: u32 = 1 << 31;
```

After:

```rust
/// The const-origin polarity at bit 31: bit *cleared* means the
/// handle was produced by `str_const!` (compile-time). This is a
/// *flag pattern*, not a non-zero mask — its value is `0` because
/// the runtime bit is the one set by `__runtime`. To test origin,
/// call `is_const()` / `is_runtime()` — do not AND against this
/// constant expecting a non-zero result on a const handle.
pub const CONST_ORIGIN_FLAG: u32 = 0;
/// Mask for the runtime-origin bit (bit 31 = 1).
pub const RUNTIME_MASK: u32 = 1 << 31;
```

No callers of `CONST_MASK` exist outside this file (verified via
workspace grep). Only the constant itself changes; nothing else
in `handle.rs` needs touching.

### Fix 3 — `StringInterner::resolve` → `Option<&str>`

**`mock/crates/hilavitkutin-str/src/interner.rs`** — change
`resolve`:

Before:

```rust
pub fn resolve(&self, s: Str) -> &str {
    if s.is_const() {
        lookup_const_by_handle(s).unwrap_or("")
    } else {
        self.arena.arena_resolve(s.id())
    }
}
```

After:

```rust
/// Resolve a handle back to a string.
///
/// Const handles delegate to the linker-section table;
/// a const-table miss returns `None`. Runtime handles always
/// resolve via the arena — an arena that cannot resolve is
/// outside the `ArenaInterner` contract (the interner hands out
/// ids it can resolve).
pub fn resolve(&self, s: Str) -> Option<&str> {
    if s.is_const() {
        lookup_const_by_handle(s)
    } else {
        Some(self.arena.arena_resolve(s.id()))
    }
}
```

**`mock/crates/hilavitkutin-persistence/src/archive_str.rs`** —
update both call sites:

`evict_str`, runtime branch:

Before: `let s = interner.resolve(handle);`
After: `let s = interner.resolve(handle).expect("runtime handles always resolve via arena");`

`inject_str`:

Before:

```rust
let resolved = interner.resolve(candidate);
if !resolved.is_empty() {
    let back = (const_fnv1a(resolved) & Str::ID_MASK as u64) as u32;
    if back == masked {

}
```

After:

```rust
if let Some(resolved) = interner.resolve(candidate) {
    let back = (const_fnv1a(resolved) & Str::ID_MASK as u64) as u32;
    if back == masked {
        return Ok(candidate);
    }
}
```

## Impl-time decisions

1. **`CONST_ORIGIN_FLAG` doc spells out the polarity.** A
   future reader who sees `const CONST_ORIGIN_FLAG: u32 = 0`
   should not be puzzled — the doc comment makes explicit that
   the flag is encoded as an *absence* of the runtime bit, and
   directs callers to `is_const` / `is_runtime`.
2. **`evict_str` runtime branch uses `.expect(...)`, not
   `.unwrap()` or `.unwrap_or("")`.** The invariant that runtime
   handles are always resolvable is a `StringInterner` contract.
   `.expect(...)` with a message documents why panicking is
   acceptable (a violated invariant, not a miss).
3. **`resolve` doc comment explicitly calls out the two paths.**
   So a caller is not surprised that the `None` case comes only
   from const misses.
4. **Per-crate descriptions over workspace-inherited.** Each
   crate's description is crate-scoped text; reads well on
   crates.io. `repository` inherits because they share the repo.
