The `morsel_size_multiplier` is basis-points-encoded (100 = 1.0x per
doc comment) — still a count semantically, so `USize` fits. Choosing

engine scaffolding uniform; the real adaptive heuristic lands in a
later round (the module is skeleton-stage per its header comment),
