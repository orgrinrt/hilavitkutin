# Source changelist: trampoline LTO survival

## CHANGE: macro emission `init_trampoline` and `shutdown_trampoline` ADD `#[inline(never)]`

File: `mock/crates/hilavitkutin-extensions-macros/src/lib.rs:170-180, 192-202`

Prepend `#[inline(never)]` to both trampoline `unsafe extern "C" fn` definitions in the `quote! { ... }` blocks. The attribute is a hint to LLVM that the function should not be inlined into call sites; it does not change behaviour but defends against an exotic LTO path that would inline the trampoline body into the descriptor's initializer expression and elide the function.

Verification: `grep -c "#\[inline(never)\]" mock/crates/hilavitkutin-extensions-macros/src/lib.rs` returns 2 (one for init, one for shutdown).
