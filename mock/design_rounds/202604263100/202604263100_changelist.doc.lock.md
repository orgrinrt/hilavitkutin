## Purpose

Second engine sub-round. Ships the stack-local resource-caching
machinery per DESIGN domain 19.

## Deliverables

- `ResourceSnapshot<R: AccessSet>` — stack-local array of resource
  values keyed by position in R. Lives on the stack of the morsel
  driver; populated before morsel loop, written back after.
- `ResourceCache<'a, R: AccessSet>` — a handle over a
  ResourceSnapshot exposing typed `get<T>()` / `set<T>()` with
  `Contains<Resource<T>>` bounds.
- `AccumulatorSlot<T>` — per-thread stack-local accumulator for
  head+tail convergence. Reset on entry, merged on exit via a
  consumer-provided combine function.
- `ConvergenceBuffer<T, const N: Cap>` — const-sized array of
  AccumulatorSlot<T> per thread.
- Pointer-separation helpers: the snapshot + cache use a different
  `NonNull<T>` newtype than column storage, to help LLVM prove
  noalias.
- Tests covering snapshot round-trip, accumulator combine, and
  pointer-separation invariants.
