- `add<WU>()` — returns self (WU type-tracked in a const array).
- `resource<T>(init: T)` — returns self.
- `column<T>()` — returns self.
- `memory(...)` / `threads(...)` / `clock(...)` — returns self.
- `build()` — returns a default-constructed Scheduler.

## Impl-time decisions

1. **Scheduler + builder = skeleton stubs** — real execution loop
   needs 5a2/5a3/5a4.
2. **Strategy selection thresholds** match DESIGN (<10K → Seq;
   depth > fibers/2 + roots ≤ 2 → Seq; roots > depth/2 → Adaptive;
   mixed → Phased).
3. **Const-generic bounds** on Scheduler + plan pass through.
4. **No real resource / column / memory / thread / clock storage**
   in the builder — all return self for now. Real wiring follows
   5a3 dispatch.
