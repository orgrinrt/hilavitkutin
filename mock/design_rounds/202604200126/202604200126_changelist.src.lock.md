## Deliverables

### 1. `src/hash.rs` — `const_fnv1a` hash

```rust
pub const FNV_OFFSET: u64 = 0xcbf29ce484222325;
pub const FNV_PRIME: u64 = 0x100000001b3;

pub const fn const_fnv1a(s: &str) -> u64 {
    let bytes = s.as_bytes();
    let mut hash = FNV_OFFSET;
    let mut i = 0;
    while i < bytes.len() {
        hash ^= bytes[i] as u64;
        hash = hash.wrapping_mul(FNV_PRIME);
        i += 1;
    }
    hash
}
```

Exposed as `pub` for hilavitkutin-persistence stable on-disk string
identity + compile-time handle derivation in `str_const!`.

### 2. `src/handle.rs` — `Str(u32)` handle

```rust
#[repr(transparent)]
#[derive(Copy, Clone, Eq, PartialEq, Hash, Debug, Default)]
pub struct Str(pub u32);

impl Str {
    pub const CONST_MASK: u32 = 0 << 31;       // bit 31 = 0
    pub const RUNTIME_MASK: u32 = 1 << 31;     // bit 31 = 1
    pub const ID_MASK: u32 = 0x0FFF_FFFF;      // bits 27-0

    #[doc(hidden)]
    pub const fn __make(id: u32) -> Self {
        Self(id & Self::ID_MASK)  // bit 31 = 0 (const by default)
    }

    pub const fn is_const(self) -> bool {
        (self.0 & Self::RUNTIME_MASK) == 0
    }

    pub const fn is_runtime(self) -> bool {
        !self.is_const()
    }

    pub const fn id(self) -> u32 {
        self.0 & Self::ID_MASK
    }

    #[doc(hidden)]
    pub const fn __runtime(id: u32) -> Self {
        Self((id & Self::ID_MASK) | Self::RUNTIME_MASK)
    }
}
```

All construction paths non-public (doc-hidden): `__make`,
`__runtime`. Consumers get `Str` via `str_const!()` or
`StringInterner::intern`.

### 3. `src/entry.rs` — `StaticStrEntry` + linker section

```rust
#[repr(C)]
pub struct StaticStrEntry {
    pub hash: Str,
    pub value: &'static str,
}
```

Collected via `#[link_section = ".hilavitkutin_strings"]`. Entries
written by `str_const!` invocations at compile time; read by the
interner at startup.

### 4. `src/macros.rs` — `str_const!()` macro

```rust
#[macro_export]
macro_rules! str_const {
    ($s:literal) => {{
        #[used]
        #[link_section = ".hilavitkutin_strings"]
        static __ENTRY: $crate::StaticStrEntry = $crate::StaticStrEntry {
            hash: $crate::Str::__make(($crate::const_fnv1a($s) & 0x0FFF_FFFF) as u32),
            value: $s,
        };
        $crate::Str::__make(($crate::const_fnv1a($s) & 0x0FFF_FFFF) as u32)
    }};
}
```

Registers entry in linker section AND returns the handle from one
macro call.

### 5. `src/section.rs` — platform-specific section walkers

```rust
/// Returns the slice of `StaticStrEntry` collected in the
/// `.hilavitkutin_strings` linker section. Returns an empty slice
/// on platforms without a walker.
pub fn static_entries() -> &'static [StaticStrEntry];
```

- Linux: `extern "C" { static __start_hilavitkutin_strings: StaticStrEntry; static __stop_hilavitkutin_strings: StaticStrEntry; }`; slice from `&__start_..&__stop_`.
- macOS: `unsafe { getsectiondata(…, "__DATA", "hilavitkutin_strings", &mut size) }`; slice from the returned pointer.
- Others (Windows, etc.): `&[]` (tracked in BACKLOG).

Uses `#[cfg(target_os = "linux")]` / `#[cfg(target_os = "macos")]`
as the switch.

## Impl-time decisions (pre-committed)

1. **Str's bit 31 = runtime, bit 31 = 0 = const.** Matches DESIGN.
2. **`__make` / `__runtime` are `pub` with `#[doc(hidden)]`**. Macros need to reach them but consumers shouldn't.
3. **`const_fnv1a` truncated to 28 bits** when used for Str construction; the full u64 hash is still exposed for persistence's use.
4. **Linker section name `.hilavitkutin_strings`** matches DESIGN + avoids clashes with other tools.
5. **`static_entries()` returns empty slice on unsupported OS** rather than panicking.
6. **StringInterner always checks const table first** even for
   `intern_static` — if the static literal was declared in a
   `str_const!` somewhere, reuse that handle.
7. **Const table lookup is linear scan**. The static table is
   usually small (<1000 entries for typical consumers). A
   follow-up round can add a perfect-hash lookup if profiling
   surfaces the scan.
