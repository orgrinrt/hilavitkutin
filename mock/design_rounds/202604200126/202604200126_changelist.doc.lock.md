## Purpose

Implement hilavitkutin-str per its existing 142-line DESIGN.md.tmpl.
Crate currently has a nuked-marker lib.rs and no other src. DESIGN
is complete — this round is mechanical fulfilment.

Shipped this round:
- `Str(u32)` handle with nibble-aligned bit layout (bit 31 const
  vs runtime; bits 27-0 = 28-bit ID).
- `str_const!()` compile-time macro with linker-section
  registration (`#[link_section = ".hilavitkutin_strings"]`).
- `StaticStrEntry` for linker-collected const string table.
- `ArenaInterner` trait (host-implemented contract).
- `StringInterner<A>` wrapper over an `ArenaInterner`, consulting
  the const linker section first.
- `AsStr` + `IntoStr` ergonomic traits.
- `const_fnv1a` hash function usable in const contexts.

Platform-specific section walking at startup:
- Linux: `__start_hilavitkutin_strings` / `__stop_hilavitkutin_strings`.
- macOS: `getsectiondata` on `__DATA,hilavitkutin_strings`.
