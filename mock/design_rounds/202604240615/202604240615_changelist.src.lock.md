## S1. Move loader source: `mock/crates/hilavitkutin-extensions/` -> `mock/crates/hilavitkutin-linking/`

`git mv` the following files from the old path to the new path, with type renames inside:

- `Cargo.toml` moves; `[package] name = "hilavitkutin-extensions"` becomes `name = "hilavitkutin-linking"`. Dependencies unchanged.
- `build.rs` moves unchanged.
- `src/lib.rs` moves. Inside: `pub struct Extension` -> `pub struct Library`; `pub use error::{ExtensionError, IncompatibilityError}` -> `pub use error::{LinkError, IncompatibilityError}`; `pub use symbol::{ExtensionSymbol, StaticRef, Symbol}` -> `pub use symbol::{LibrarySymbol, StaticRef, Symbol}`; `impl Extension` -> `impl Library`; `impl Drop for Extension` -> `impl Drop for Library`; method body updates to match renamed types. `Extension::load` / `::resolve` / `::resolve_static` / `::close` become `Library::load` / `::resolve` / `::resolve_static` / `::close` by construction.
- `src/error.rs` moves. Inside: `pub enum ExtensionError` -> `pub enum LinkError`. `IncompatibilityError` unchanged. Method / variant names and fields unchanged.
- `src/symbol.rs` moves. Inside: `pub trait ExtensionSymbol` -> `pub trait LibrarySymbol`. Lifetime parameter `'ext` renames to `'lib` on `Symbol<'ext, T>` / `StaticRef<'ext, T>` and their impl blocks. The `impl_extension_symbol_for_fn!` macro renames to `impl_library_symbol_for_fn!`.
- `src/backend/mod.rs`, `src/backend/unix.rs`, `src/backend/windows.rs` move unchanged (internal FFI; no public types involved).
- `tests/basic.rs` moves. Inside: replace `Extension::` with `Library::`, `ExtensionError::` with `LinkError::`, `hilavitkutin_extensions` crate path with `hilavitkutin_linking`. Test function names unchanged. Expected behaviours unchanged.
