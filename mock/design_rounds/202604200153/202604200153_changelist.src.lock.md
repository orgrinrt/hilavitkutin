## Impl-time decisions (pre-committed)

1. **`PragmaSet` uses u16 bitmask** — 13 variants fit with room
   for growth. `ParallelCodegen(u8)` stored separately since it
   carries a param.
2. **Profile::from_cargo_profile unknown-fallthrough = Dev** — safe
   default; unknown profile names happen only in exotic setups.
3. **`TargetAxis`, `TierAxis`, `PassesAxis` as enums** — small
   and finite; don't need Cap-style newtypes this round.
4. **`bootstrap_from_buildscript` is a stub** that emits
   arvo_fast_math cfg when FastMath is present. The full rustc
   wrapper + config.toml schema + LLVM plugin dylib linking
   are follow-ups.
5. **`REQUIREMENTS` table is `const`** — static data. When a
   follow-up round adds runtime-detection code, it consults this
   table statically.
6. **`guards.rs` macro** stays minimal — a single `require_isa!`
   this round; vendor / arch guards land with the wrapper-script
   round when the flag-mapping table is authored.
