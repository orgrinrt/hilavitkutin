# Changelist (src): extensions cop-out cleanup

**Date:** 2026-04-24
**Phase:** SRC
**Source topics:** `202604500000_topic.extensions-cop-out-cleanup.md`
**Pairs with:** `202604500000_changelist.doc.lock.md`

## Src-side changes

### `mock/crates/hilavitkutin-extensions/src/descriptor.rs`

- Switch the import to include `CStr`: `use core::ffi::{c_void, CStr};`.
- Introduce `AbiVersion` newtype: `#[repr(transparent)] #[derive(Copy, Clone, PartialEq, Eq)] pub struct AbiVersion(pub u32);`. Transparent so the FFI wire layout is identical to a plain `u32`. The `pub u32` field mirrors the existing `CapabilityId(pub u64)` pattern.
- `HOST_ABI_VERSION: u32 = 1` becomes `HOST_ABI_VERSION: AbiVersion = AbiVersion(1)`.
- `DESCRIPTOR_SYMBOL: &[u8] = b"__hilavitkutin_extension_descriptor\0"` becomes `DESCRIPTOR_SYMBOL: &CStr` constructed via `unsafe { CStr::from_bytes_with_nul_unchecked(b"__hilavitkutin_extension_descriptor\0") }`. Const-safe because the literal's trailing nul and lack of interior nul are compile-time facts; a `// SAFETY:` comment records the invariant.

### `mock/crates/hilavitkutin-extensions/src/error.rs`

- Update import from `descriptor` to include `AbiVersion`.
- `AbiVersionMismatch { expected: u32, got: u32 }` becomes `AbiVersionMismatch { expected: AbiVersion, got: AbiVersion }`. Same wire bits (`AbiVersion` is `#[repr(transparent)]` over `u32`), host-facing type gets the semantic name.

### `mock/crates/hilavitkutin-extensions/src/host.rs` (and any call site)

- Host-side comparison: wrap `descriptor.abi_version` into `AbiVersion` at read time before comparing with `HOST_ABI_VERSION`, propagate both as `AbiVersion` into `ExtensionError::AbiVersionMismatch`. `ExtensionDescriptor.abi_version: u32` stays bare on the FFI side.
- Any `Library::resolve_static(DESCRIPTOR_SYMBOL)` call signature adjustment: the linking-layer resolve function takes `&CStr` in place of `&[u8]` if the prior signature used the byte slice. If the linking layer's signature changes are deeper, those land with #206 or a sibling round.

## What this changelist does not touch

- FFI wire surfaces: `ExtensionDescriptor.abi_version: u32`, `ExtensionVersion { major: u16, ... }`, `CapabilityId(pub u64)`, `ExtensionMeta::NAME`. Out of scope per topic; #206 scope.
- `hilavitkutin-linking::Library::resolve_static` signature change beyond accepting `&CStr` (if already accepted, no change).
