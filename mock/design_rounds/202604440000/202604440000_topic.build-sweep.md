## Character of each site

- **pragma.rs** stores 13 pragma variants as a `u16` bit-mask (3
  bits of headroom), plus a separate `u8` unit count for
  `ParallelCodegen(n)`. Widths are algorithmic — changing them
  changes the bit layout.
- **pragma.rs `PragmaIter::next`** implements `std::iter::Iterator`;
  `Option<Pragma>` is the required return per the trait. Std-trait
  conformance supersedes the Maybe flip.
- **config.rs `target: Option<TargetAxis>`** — a pub field; the
  cargo env may lack the `TARGET_FEATURE` var, yielding no
  target-axis info. Could flip to `Maybe<TargetAxis>`; keeping
  Option here because `BuildConfig` is consumer-visible and
  matches `std::env::var` return semantics.
- **`&str` inputs** (config.rs, profile.rs) — cargo env values
  arrive as `String`, consumers pass `.as_str()` or string
  literals. `&str` is the cargo-side boundary.
- **`bool` returns** (`fast_math`, `contains`, `same_variant`) —
  predicates. Flipping to `Bool` adds `.0` at every caller and
  provides no extra safety in a build-time crate whose output
  flows into `build.rs` stdout.
